# GenkiFlow

A Japanese study app for iOS and macOS with spaced-repetition review sessions, a built-in dictionary, and streak tracking.

## Requirements

- Xcode 26+
- iOS 26.2+ / macOS 26.2+

## Building

Open `GenkiFlow.xcodeproj` in Xcode and run, or build from the command line:

```bash
xcodebuild -project GenkiFlow.xcodeproj -scheme GenkiFlow -destination "generic/platform=iOS" -configuration Release
```
